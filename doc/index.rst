Examples
========

Here are some examples how to use ``openstreetmap`` directive.

.. list-table:: ``openstreetmap`` directive examples
   :header-rows: 1

   * - Example
     - Description
   * - `No label <examples/no_label.html>`_
     - It shows usage of label.
   * - `No marker <examples/no_marker.html>`_
     - It just shows simple map only.
   * - `No renderer <examples/no_renderer.html>`_
     - It shows usage of ``:renderer:`` option.
   * - `Multiple markers <examples/multiple_markers.html>`_
     - It shows multiple markers on map.
   * - `Location for map <examples/location_map.html>`_
     - It shows usage of ``location:`` for map.
   * - `Location for marker <examples/location_marker.html>`_
     - It shows usage of ``location:`` for marker.
