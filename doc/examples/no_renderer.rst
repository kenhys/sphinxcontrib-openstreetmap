
No renderer example
-------------------

Here is the simple markup which shows map::

  .. openstreetmap::
     :id: example_openstreetmap1
     :latitude: 40.689249
     :longitude: -74.0445

This markup generates following map.

.. openstreetmap::
   :id: example_openstreetmap1
   :latitude: 40.689249
   :longitude: -74.0445
