
Default label example
---------------------

Here is the simple markup which shows map::

  .. openstreetmap::
     :id: default_label
     :location: 40.689249,-74.0445

This markup generates following map.

.. openstreetmap::
   :id: default_label
   :location: 40.689249,-74.0445
